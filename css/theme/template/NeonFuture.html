Introduction

Welcome to Neon Future, a cyberpunk-inspired template for Reveal.js presentations.

This template features:

  • Vibrant neon colors on dark backgrounds
  • Smooth animations and transitions
  • Custom styled elements
  • Fully responsive design

Key Features

Modern Design

Cyberpunk aesthetic with glowing elements and futuristic feel

Fully Customizable

Easy to modify colors and styles through CSS variables

Code Highlighting

Beautiful syntax highlighting for code examples

Responsive Layout

Looks great on desktop, tablet, and mobile devices

Code Example

Here's how to initialize Reveal.js with this template:


// Initialize Reveal.js
Reveal.initialize({
    controls: true,
    progress: true,
    center: true,
    hash: true,
    
    // Theme options
    transition: 'slide',
    backgroundTransition: 'fade',
    
    // Custom options for Neon Future
    neonColors: {
        primary: '#0ff0fc',
        accent: '#ff2d95'
    }
});

Custom Elements

This template includes special CSS classes for emphasis:

.neon-highlight - For important text

.neon-box - For content containers

Perfect for highlighting important information or grouping related content.

Get Started

To use this template in your Reveal.js presentation:

  1. Copy the CSS from the neon-future.css file
  2. Add it to your Reveal.js project
  3. Customize the colors and styles as needed
  4. Enjoy your cyberpunk-themed presentation!
