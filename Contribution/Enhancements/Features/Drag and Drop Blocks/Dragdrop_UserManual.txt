Drag-and-Drop Blocks — User Manual (CDN Version)

Feature
Movable blocks inside a canvas area, allowing diagrams or workflows to be created live.

How to Use
1) Open this HTML file in any browser. It works anywhere because Reveal.js loads from CDN.
2) Go to the Drag-and-Drop slide.
3) Click and drag any block (Start, Process, Decision, Result) to move it within the canvas.
4) Rearrange blocks to create a flow or mind map.

Advantages
• Enables interactive brainstorming during a presentation.
• Useful for teaching workflows or processes.
• CDN links guarantee the demo works in any folder.
